How to run the flask application/web server.
assuming that you already have flask and python installed on your computer.

1. Open windows cmd.

2. Navigate to the directory that hold the correct app directory(Flask) -- NOT TO Flask Directory!.

3. Press  "FLASK_APP=Flask"

4. Press "FLASK_ENV=development"

5. If it is the first time on this computer so you need to initialize the database:
	Press "flask init-db"

6. press "flask run"

7. Copy the URL that you got and enjoy (https://127.0.0.0/5000)
